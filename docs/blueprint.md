# **App Name**: CampusConnect

## Core Features:

- Student Enrollment: Allows the secretary to register students with personal details, parent information, class assignments, fee payments, and refectory table assignments.
- Role-Based Access Control: Implements role-based access for different users (secretary, patron, matron, admin) to restrict access based on their roles.
- Refectory Management: Tracks refectory table assignments for both boarding and external students, with time-based restrictions for external students.
- Utility Tracking: Enables patrons and matrons to monitor and manage student utilities and material status using the data provided.
- Material List: Patron and Matron has access to materials listing module.
- Admin Dashboard: Provides a centralized dashboard for administrators to manage data, activities, and user assignments.
- Work Assignment tool: Allows admin, patron, and matron roles to LLM powered assistant, helping users create daily to-do lists.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to evoke trust, stability, and knowledge.
- Background color: Light blue-gray (#E8EAF6), offering a clean and calm visual space.
- Accent color: Vivid orange (#FF9800) to highlight critical actions and information.
- Body and headline font: 'Inter' for a modern, neutral, and highly readable interface.
- Use simple, clear icons to represent actions and categories, enhancing usability.
- Design a clean, intuitive layout for easy navigation, focusing on essential data display.
- Incorporate subtle transitions to enhance user experience without distracting from the main tasks.
